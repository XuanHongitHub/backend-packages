# Introduction for Antigravity

Chào **Antigravity**, đây là dự án **Bepack Laravel** - một bộ công cụ Backend toàn năng. Để làm việc hiệu quả nhất, bạn cần nắm vững:

## 1. Stack công nghệ
*   **PHP 8.3+** & **Laravel 11**.
*   **Filament v4** (Trọng tâm của Admin Panel).
*   **Livewire v3** & **Alpine.js**.
*   **Tailwind CSS**.

## 2. Cách bạn nên tiếp cận dự án
*   **Đọc kỹ Specs:** Trước mỗi Module, hãy đọc file tương ứng trong `docs/implementation/`.
*   **Tuân thủ Rules:** Luôn đối soát công việc với `docs/ai_agent_rules/`.
*   **Tư duy Sản phẩm:** Đừng chỉ viết code cho xong task, hãy nghĩ xem User sẽ dùng tính năng này như thế nào cho "sướng" nhất.

## 3. Cấu trúc quan trọng đặc thù
*   Dự án sử dụng **Actions Pattern**: Mọi logic nghiệp vụ nằm trong `app/Actions`.
*   Dự án sử dụng **Settings Pattern**: Cấu hình nằm trong `app/Settings`.
*   Dự án sử dụng **Media Library**: Quản lý file qua Spatie Media Library.

## 4. Mục tiêu cuối cùng
Tạo ra một bản cài đặt Laravel sạch (Clean), đẹp (Premium UI), và thông minh (Smart Logic) để có thể nhân bản cho hàng trăm dự án sau này.
